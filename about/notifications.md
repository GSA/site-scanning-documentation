The Site Scanning team operates a notification-only listserve that developers can subscribe to in order to receive notifications about changes to the static data files and the API endpoint, with a particular focus on alerting developers to potential breaking changes.  Specifically, alerts will be sent out for: 
- Column headers changing in static files
- Column order changing in static files 
- Static data file format or structure changing
- URLs of file downloads changing
- API design changing

To sign up, email site-scanning@gsa.gov and ask to be added.
